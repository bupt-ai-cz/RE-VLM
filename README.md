## Overview

**RE-VLM** is the **first dual-stream vision-language model** that jointly leverages **RGB images** and **event streams** for robust scene understanding across both normal and challenging conditions.

Traditional Vision-Language Models (VLMs) suffer from severe performance degradation when RGB images are captured under adverse conditions (low light, high dynamic range, high-speed motion). RE-VLM addresses this fundamental limitation by incorporating event camera data, which captures pixel-level brightness changes with microsecond latency and high dynamic range.
